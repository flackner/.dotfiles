# .dotfiles

Neovim is installed from the official pre-built Linux archive. JetBrainsMono Nerd Font is downloaded automatically from the Nerd Fonts v3.4.0 release archive by the installer scripts.

install with:
```
./installSuse.sh
# or
./installUbuntu.sh

./doStow.sh
```

update neovim with:
```
./updateNeovim.sh
```
